### Présentation du moteur

raccourcis : 

- avance/recul caméra: Souris bouton **gauche + drag**
- rotation : souris bouton **droit + drag**
- Déplacement clavier : bouton **souris + W, A, S, D, Q**
- Focus sur un objet : sélectionner l'objet puis **Touche F**
- Déplacement autour d'un objet : (style maya) Sélectionner puis **ALT** + souris clic gauche/droit + drag



Trigger : 

- clic droit -- addEvent -- overlapp  actor
- Blueprint -> print
